# ts-express-typeorm

TypeScript & Express.js & TypeORM Example

## Dependencies

> nodejs >= 12.14.0

- `tsyringe` (dependency injection)
- `express`
- `typeorm`
- `class-validator`
- `morgan`
- `pino`
- `jest`(`ts-jest`)

## Linting & Formatting

- `typescript-eslint`
- `prettier`

## LiveReload for development

> TypeScript >= 3.5.0

`concurrently` & incremental build & `nodemon`

## Run

`./run-database.sh && pm2 start ecosystem.config.js`
